# Javascript Hello World

A simple static website demonstrating:
- Light/Dark mode toggle with persistent preference (localStorage)
- Modular, maintainable HTML/CSS/JS structure
- Multi-page navigation (Home, Articles, About)

## Features
- Responsive design
- Accessible navigation
- Easy to extend and customize

## Getting Started
1. Clone the repository:
   ```
   git clone https://github.com/oafleming/Javascript-Hello-World.git
   ```
2. Open `index.html` in your browser.
